# Word Saver Chrome 插件

一个简单的英语单词收藏插件：
- 在网页中选中英文单词，右键点击“保存单词”
- 插件优先使用有道翻译获取中文释义（失败时自动回退 Google）并保存到本地
- 点击保存后会立即在当前网页右下角显示该单词释义
- 点击浏览器工具栏中的插件图标，在“单词本”里查看、搜索、删除或清空单词
- 单词本支持记忆模式（显示单词+释义）
- 单词本支持背词模式（仅单词 / 仅释义），并且每轮顺序随机打乱
- 背词模式每个词条都有“显示单词/显示释义”按钮，支持逐词揭示答案
- 词条操作按钮使用紧凑图标（显示/删除），减少占用空间
- 单词列表固定分页：每页 10 个
- 搜索框按需弹出，不常驻显示

## 安装方式（开发者模式）
1. 打开 Chrome，进入 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录：`D:\WorkDirQiax\VulnDiscov\word_plugin`

## 使用说明
1. 在任意网页中选中一个英文单词（如 `ability`）
2. 右键菜单点击“保存单词”
3. 插件会在网页中显示该词释义
4. 点击浏览器右上角插件图标，打开单词本查看已保存内容

## 数据存储
- 使用 `chrome.storage.local` 本地存储
- 存储字段包含：
  - `word`：英文单词
  - `meaning`：中文释义
  - `pageUrl`：来源网页链接
  - `createdAt`：保存时间（ISO）

## 说明
- 中文释义优先通过 `https://dict.youdao.com/jsonapi` 获取，失败时回退到 `https://translate.googleapis.com`
- 释义显示优先从本地读取（已保存单词本 + `translationCache` 缓存），本地没有才请求翻译接口
- 翻译结果会缓存到本地（`translationCache`），同一单词后续保存不会重复请求翻译接口
- 若网络不可达或接口不可用，会提示“翻译服务不可用，请稍后重试”
